Authentication Error

What happened?

Your request could not be authenticated. This could be due to one of the following reasons:

  • Your authentication token has expired
  • Your token is invalid or malformed
  • The server's authentication key has been rotated
  • You did not include an authentication token in your request

How to fix it

Here are some steps you can take to resolve this issue:

  1. Log out and log back in to get a fresh authentication token
  2. Make sure your request includes an Authorization header with the format Bearer your-token-here
  3. Check that your token hasn't expired (JWT tokens typically expire after a certain period)
  4. If you're a developer, verify that the server's JWT secret is correctly configured

If you continue to experience issues, please contact support with the time and details of your request.
